#  ✨HOW TO START✨
## 1️⃣ you need to create a `.env` file and add there `DB_URL = 'your_database_url'` 
## 2️⃣ then you need to install the dependencies using the command `npm install` 
## 3️⃣ after that, you can run the project using the command `node db.js` or `node sequelize.js` depending on which file you want to run.
